NetGauge: End of Test Configuration

Team Ookla - November 11, 2016 19:21

 

In NetGauge, the end of test panel in which a users test results are presented is entirely configurable. There is a tile-based system controlled by a simple configuration method which allows you to easily customize the panel according to your preferences.

The grid is 5x2, but it's possible to reduce the number of columns as desired.

Tile sizes

Five different tile sizes are available: 1x1, 1x2, 2x1, 2x2, 4x2

Here is an example of the 3 title sizes available for the Packet Loss test.

Tile types

Five tile types are available:

  • Speed (S)  - 2x2, 4x2
  • Latency (L) - 1x1, 1x2, 2x1
  • Jitter (J) - 1x1, 1x2, 2x1
  • Packet loss (P) - 1x1, 1x2, 2x1
  • Firewall (F) - 1x1, 1x2, 2x1

 

Configuration examples

4x2 Full Test Grid

<endoftest columns="4"
    toprow="LPSS" 
 bottomrow="FFSS" />

 

5x2 Full Test Grid

<endoftest columns="5"  
    toprow="LPFSS"  
 bottomrow="JPFSS" />

 

5x2 Speed Test Grid

<endoftest columns="5"  
    toprow="LSSSS"  
 bottomrow="LSSSS" />
