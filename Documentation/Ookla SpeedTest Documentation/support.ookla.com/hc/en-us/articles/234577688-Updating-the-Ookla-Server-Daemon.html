Updating the Ookla Server Daemon

Brian Turley - January 05, 2018 19:54

Occasionally the Ookla Server Daemon will require an update. For help determining your current version, please see How do I determine the version of my NetGauge OoklaServer Daemon?.

Updating the daemon is a fairly straightforward process. Please follow the instructions for your operating system below.

Linux

  • Open a terminal window in the directory containing your existing Ookla Server Daemon instance.
  • Run the install script to download and install the latest version: 
    ./ooklaserver.sh install
  • The script will now determine your server version and prompt you if you would like to install the new version. Answer yes (y) and the daemon will install and automatically start.

Windows

  • Open a command prompt as administrator.
  • Stop any existing instance of the daemon with the following command: 
    >taskkill /IM OoklaServer.exe
  • Download the most current version of the server daemon here:
    • Windows.zip
  • Extract the contents of Windows.zip to the directory containing your existing version of the server daemon and overwrite any existing files.
  • If OoklaServer.exe was previously registered as a service, restart the service with the following command: 
    >net start OoklaServer
  • If OoklaServer.exe is not registered as a service, simply run the following: 
    >OoklaServer.exe
