NetGauge: Client Error Messages - Licensing

Team Ookla - November 11, 2016 19:21

Licensing errors can occur for any of the following reasons:

Error Message Title    Error Message
Missing License Key

The license for this Speed Test is missing. Please contact the webmaster of this site to let them know.
Configuration Load Failed   

The configuration for the Speed Test couldn't be loaded. Please contact the webmaster of this site to let them know.
Expired License

The license for this Speed Test has expired. Please contact the webmaster of this site to let them know.
Wrong License

This Speed Test is not setup to work at this URL. Please contact the webmaster of this site to let them know.
Invalid License

The license for this Speed Test is invalid. Please contact the webmaster of this site to let them know.

 

Resolution and Troubleshooting:

Error Message Title    Resolution
Missing License Key

This means there was no license key found in the settings.xml file. If a custom extension (ex: php, asp, aspx, jsp) is used, it would be defined as configExtension="php" within the index.html as a flashvar. If set correctly, there could be an issue with the scripting language running properly on the web server. 
Configuration Load Failed   

This means the test was unable to load the settings file itself. Make sure that you can load it directly in your browser without an error. If a custom extension (ex: php, asp, aspx, jsp) is used, it would be defined as configExtension="php" within the index.html as a flashvar. If set correctly, there could be an issue with the scripting language running properly on the web server.
Expired License

The license key that has been loaded by NetGauge (found in the settings file on the server) is not valid due to the expiration date being passed. You will need to contact Ookla Support to make sure that your account is current and you have your most recent license key
Wrong License

The license key that has been loaded by NetGauge (found in the settings file on the server) is not valid due to the URL netgauge.swf was loaded from not being supported or in the access list. Make sure that the full path to netgauge.swf matches what Ookla has setup for your account.
Invalid License

The license key that has been loaded by NetGauge is not valid for this application. This is usually caused by a typo or hidden character in the license key.
