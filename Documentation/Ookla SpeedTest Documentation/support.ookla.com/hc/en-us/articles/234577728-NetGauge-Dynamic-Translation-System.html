NetGauge: Dynamic Translation System

Team Ookla - November 11, 2016 19:21

Translation can be configured from the settings.xml file. You can add multiple languages and modify them as desired. You can even modify the default text within NetGauge.

To do so, you simply incorporate a translation block to your configuration file. You can use it to add new translations or selectively modify existing text values. For each desired language, add an additional translation block.

For example, to replace "Latency" with "Ping" in your test, add the following to settings.xml:

<translation lang="en">
	<text id="label_latency">Ping</text>
	<text id="testinglatency">Testing Ping</text>
</translation>

The default language used by NetGauge is English. Adding a translation block for English will replace the default text within NetGauge with the text included in your translation block. To use a language other than English, a language parameter must be passed to the Flash object as it loads. This is accomplished using the existing swfobject JavaScript library reference in the default index.html file included with NetGauge.

To load a test in a specific language, we can add a static entry to the index.html file. For example, to force a test to load in Spanish regardless of the user’s location, modify the existing netgauge.swf file reference from

swfobject.embedSWF("netgauge.swf?v=3.0", "flashcontent", "620", "420", "10.0.0", false, flashvars, params, attributes);

to equal

swfobject.embedSWF("netgauge.swf?v=3.0&lang=es", "flashcontent", "620", "420", "10.0.0", false, flashvars, params, attributes);

To automatically load a test in the user’s default browser language, we will need to add a variable and detect the language. Within the default index.html, modify the existing netgauge.swf file reference from

swfobject.embedSWF("netgauge.swf?v=3.0", "flashcontent", "620", "420", "10.0.0", false, flashvars, params, attributes);

to equal

var myLang = (window.navigator.userLanguage || window.navigator.language).substring(0,2);

swfobject.embedSWF("netgauge.swf?v=3.0&lang=" + myLang, "flashcontent", "620", "420", "10.0.0", false, flashvars, params, attributes);

Note: If the detected language does not have a translation block present in settings.xml, NetGauge will default to using English.
