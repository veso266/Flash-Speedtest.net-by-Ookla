What fields are included in Speedtest net data?

Saoud - March 01, 2017 19:33

The table below represents the Speedtest Intelligence Data available from users testing directly to Speedtest.net from a client web browser such as IE, Chrome, Safari, or FireFox. The raw data set provides a means to gain superior knowledge about various wireline services including Cable, DSL, 3G/4G Dongles, and Fiber networks with the accuracy and unparalleled volume of Ookla's authoritative Speedtest.net.

Field Name Description Data Type
test_id unique identifier of test number
test_date YYYY-MM-DD HH:MM:SS (defaults to Pacific Time but Ookla can accommodate different timezones) datetime
client_ip_id unique ip address identifier of client number
download_kbps download speed in kilobits per second number
upload_kbps upload speed in kilobits per second number
latency_ms ping in milliseconds number
server_name name of server tested to (name of city it is located in) text
server_country country name of server tested to text
server_country_code country code of server tested to text
server_latitude latitude of server tested to number
server_longitude longitude of server tested to number
server_sponsor_name  name of server sponsor text
user_agent http://en.wikipedia.org/wiki/User_agent text
isp_name Internet Service Provider of client text
client_net_speed 0-Unknown, 1-Dialup, 2-Cable/DSL, 3-Corporate, 4-Cellular number
is_isp 1 = is ISP, 0 = is not number
client_country country name of the client text
client_country_code country code of the client text
client_region region name of client (this will be a state in the United States) text
client_region_code region code of client text
client_city city of client text
client_latitude latitude of client number
client_longitude longitude of client number
miles_between miles between the client and the server tested to number
