NetGauge: Dynamic Translation System - Error Messages

Team Ookla - November 11, 2016 19:21

Translation can be configured from the settings.xml file. You can add languages and modify them as desired. This article applies only to NetGauge error messages. For more information about translation in NetGauge, please see the article NetGauge Dynamic Translation System.

To do so, you simply incorporate a translation block to your configuration file. You can use it to add new translations or selectively modify existing translations.

For example, to replace the error message "Missing License Key" with "he license for this Speed Test is missing." in your test, add the following entry to settings.xml:

<translation lang="en">
  <text id="license06title">Missing License Key</text>
  <text id="license06messsage">The license for this Speed Test is missing.</text>
</translation>
