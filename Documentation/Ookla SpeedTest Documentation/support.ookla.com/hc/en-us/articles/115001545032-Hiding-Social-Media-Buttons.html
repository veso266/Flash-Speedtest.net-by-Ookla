Hiding Social Media Buttons

Karthik - September 13, 2017 05:52

With Speedtest Custom, it is possible to minimally customize the UI.  For example, if sharing results through the Social Media buttons needs to be disabled, it can be done through Custom CSS.  To learn more about this feature, read our Configuring Your Test Using Custom CSS article.

When a test is completed in Speedtest Custom, it gives end users an option to share the results to social media or copy the link to share it to any one via email, Instant Messaging client, etc.

This option can be disabled so that the result data can be confidential.  

1. Go to Configuration tab and click on the Interface sub tab.

2. Type the CSS snippet in the text box below "Custom CSS" heading:

.share__links {
     display: none;
}

3. Click on "Preview CSS" to verify if the CSS code works as expected in Speedtest Custom.

4. Once Preview looks satisfactory, click on "Save & Publish" button to save the changes.

The above is an example to disable all social media buttons.  It is possible to hide specific social media through Custom CSS.  Browser tools such as Chrome Developer tools can be used to scan the CSS selectors.  Please note that using too much CSS code may affect the Speedtest Custom's page load time and also affect cross browser functionality.
