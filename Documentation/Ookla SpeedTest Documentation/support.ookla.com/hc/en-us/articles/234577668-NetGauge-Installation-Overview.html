NetGauge: Installation Overview

Brian Turley - November 11, 2016 23:12

This article provides a general overview for the installation of NetGauge. For more detailed information regarding installation and configuration, please visit the landing page within our knowledge base for NetGauge. For those upgrading their Line Quality Test or Speedtest deployments, please see Upgrade from Speedtest or Line Quality Test for notes on upgrading.

 

Review Minimum Requirements

  1. Verify your server hardware meets or exceeds the specifications listed in the Server & Client Requirements article.
  2. Verify that your web server is accessible and able to serve content prior to installing NetGauge.
    • You should be able to open a web browser and navigate to your sever's URL or IP address and receive a response.

 

Add Client Content

  1. Download the required client facing content to your web server.
    • Licensed Clients - Your account executive will provide you with a direct link to your specific, pre-configured installation package.
    • Trial Users - After completing registration, you will be provided a link to download the trial installation package which contains an Ookla branded trial template.
  2. Install the client facing content on your web server.
    • Extract the downloaded archive into your root web directory, or a subfolder, such as /netgauge.
    • Licensed Clients - this location is generally defined by the license URL.
    • Trial users - You will need to edit the host URL in the settings.xml file. Within the extracted folder, open settings.xml. Replace the existing url within the line <method type="tcp" host="mtl.speedtest.net:8080"/> with your desired server’s URL or IP address. Make sure to leave the port number in place. When finished, the setting should look similar to either <method type="tcp" host="yourserver.com:8080"/> or <method type="tcp" host="192.168.1.100:8080"/>
    • Depending on your preferred web server, the required steps for installation will vary. For detailed instructions please see the KB article NetGauge: Client Installation & Configuration

 

 

Install the Server Daemon

  1. Download and install the required server daemon
    • Windows - Follow the instructions contained in KB article NetGauge: Windows Daemon Setup (Registered Service)
    • Linux/OS X - Follow the instructions contained in KB article NetGauge: Server Daemon Setup

 

 

Verify Configuration

  1. Verify the server daemon is running and listening by running telnet from a secondary client computer as detailed in the KB articles referenced in the server daemon section above
  2. If you are unable to telnet or are experiencing any other configuration problem, please see the troubleshooting articles here: Knowledge Base/NetGauge/Troubleshooting
